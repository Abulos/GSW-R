The stable version of gsw is provided on the R archives. It is installed from within R, just like other packages.

Users who are set up to build packages (which requires a C compiler, etc) may stay at the cutting edge by installing a development version. This may be done as follows, within an R session.

{% highlight R %} library(devtools) install_github("TEOS-10/GSW-R", "master") {% endhighlight %}
