Gsw is a package for the R statistical language that helps Oceanographers do their work by providing an interface to the TEOS-10 Gibbs SeaWater library.

Gsw may be used alone, or within the oce package.

The latest official release of Gsw is available on CRAN, and users may follow the progress (or contribute to) the development process on github.com.

The authors would appreciate a citation in papers that use Gsw. For any given version, the citation information may be found with {% highlight R %} citation("gsw") {% endhighlight %} For example, the latest version should be cited as Dan Kelley, Clark Richards and WG127 SCOR/IAPSO (2015). gsw: Gibbs Sea Water Functions. R package version 1.0-3. http://teos-10.github.io/GSW-R/index.html, or entered into BibTeX as

  @Manual{gswpackage,
    title = {gsw: Gibbs Sea Water Functions},
    author = {Dan Kelley and Clark Richards and WG127 SCOR/IAPSO},
    year = {2015},
    note = {R package version 1.0-3},
    url = {http://teos-10.github.io/GSW-R/index.html},
  }
